 # To do list 

 This simple to do list app was created by following Brad Traversy's React Crash Course 2019. 

 Check out the live demo [here](https://peaceful-thicket-10418.herokuapp.com/).


 ## Technologies utilized
 1. ReactJS
 1. JSONPlaceholder
 1. Axios
 1. Heroku
